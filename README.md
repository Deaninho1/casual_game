# casual_game

Ik kies extra fun

Mijn core mechanic is dat de speler obstakels moet ontwijken net zoals in frogger en naar de andere kant moet komen.

je doet dit door te klikken en swipen.

De speler probeert om een zo hoog mogelijke highscore te halen zonder dood te gaan.

Het wordt uitdagend omdat er steeds nieuwe en moeilijkere obstakels bij komen.

De actie blijft leuk om te herhalen omdat je graag een zo hoog mogelijke highscore wil hebben.

Ik maak deze actie satisfying door combos en close calls op het scherm te zetten met animaties.

### I'm a mechanic
-------

Je kan swipen om je poppetje te bewegen dit kan naar voren, achter, links en rechts.
Elke keer als je swiped hoor je een spring geluid. Je kan verschillende power ups en munten oppakken die ook met hun eigen geluid komen.
als je een nieuwe highscore haalt komt er groot tekst op beeld dat je een nieuwe highscore hebt. Ook als je een close call hebt (bijna geraakt wordt) wil ik grote tekst op het beeld en extra punten. Ik wil ook een close call combo systeem voor extra punten en een reden en een aanmoediging op risky te spelen.

Je hebt natuurlijk ook auto's en treinen waar je voor moet oppassen ook wil ik aan deze elementen en toeter effect toevoegen.Ik wil ook nog kleine animaties aan de speler toevoegen als je beweegt.
