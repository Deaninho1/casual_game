# casual_game

Ik kies extra fun

Mijn core mechanic is dat de speler obstakels moet ontwijken net zoals in frogger en naar de andere kant moet komen.

je doet dit door te klikken en swipen.

De speler probeert om een zo hoog mogelijke highscore te halen zonder dood te gaan.

Het wordt uitdagend omdat er steeds nieuwe en moeilijkere obstakels bij komen.

De actie blijft leuk om te herhalen omdat je graag een zo hoog mogelijke highscore wil hebben.

Ik maak deze actie satisfying door combos en close calls op het scherm te zetten met animaties.
